# 创建规则

![规则创建与绑定流程图](/illustrations/rules-workflow.svg)

签到规则决定系统在签到时使用哪套资料和偏好。规则创建好后，可以重复绑定到多门课程。

## 什么时候需要创建规则

- 你要做位置签到，需要提前保存坐标和地址
- 你要做拍照签到，需要准备默认图片
- 你要做人脸签到，需要填写人脸照片链接
- 你想给不同课程使用不同延迟、不同签到类型范围

## 新建规则步骤

1. 打开“签到规则”。
2. 点击“新建规则”。
3. 填写规则名称。
4. 选择适用的签到类型。
5. 按需补充位置、图片、来源和其他偏好。
6. 保存规则。

## 规则里常用的配置项

### 规则名称

建议按地点或用途命名，例如：

- 教学楼 A305
- 宿舍备用规则
- 早八课程默认规则

### 适用签到类型

你可以控制这条规则用于哪些签到类型。常见做法：

- 想通用一点：保持全选
- 只做位置签到：只保留位置相关类型
- 只做拍照或人脸：按需要单独勾选

### 签到延迟

适合不想“秒签”的场景。填入秒数后，系统会按这个延迟执行。

### 签到人数阈值

适合希望更稳一点再签到的场景。想尽早签到就填小一点，想更保守就填大一点。

### 群组签到

同一组人共用一套坐标时可以打开，方便多门课复用同一地点设置。

### 默认规则

没有单独绑定规则的课程，会优先使用默认规则。新用户建议至少保留一条默认规则。

### 自动签退

如果课程包含签退环节，打开后更省事；关闭后需要你自己留意签退任务。

### 签到来源

- `全部`：都处理
- `仅课程`：只处理课程页里的签到
- `仅群聊`：只处理群聊来源的签到

## 位置签到怎么配

1. 在规则编辑页打开地图选点。
2. 选择目标位置。
3. 确认地址、经纬度是否正确。
4. 保存规则。

如果你已经知道经纬度，也可以直接手填。

## 拍照签到怎么配

1. 先准备一张可直接访问的图片链接。
2. 粘贴到“拍照签到默认图”。
3. 保存规则。

这样遇到拍照签到时，就会优先使用这张图。

## 人脸签到怎么配

1. 先准备一张你自己的人脸图片直链。
2. 粘贴到“人脸照片链接”。
3. 保存规则。

人脸图片建议单独准备，避免和普通拍照签到图片混用。

## 分享与复用

“我的规则”支持两种复用方式：

- 公开分享：让其他人能在分享中心看到并使用你的规则
- 邀请链接或规则 ID：点对点发给同学，一键加入

适合分享的内容是规则结构和地点设置；涉及你个人图片的内容建议自己单独检查后再分享。

::: tip 最稳妥的做法
先把一条规则调通，再复制思路去做其他规则，不要一上来同时建很多条。
:::
