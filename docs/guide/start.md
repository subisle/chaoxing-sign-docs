# 快速开始

![快速开始流程图](/illustrations/hero-dashboard.svg)

## 使用前准备

- 你需要一个可正常登录的学习通账号
- 你需要能访问本项目的前端页面
- 如果页面开启了人机验证，按页面提示完成验证即可

## 第一次使用

1. 打开登录页。
2. 输入学习通手机号和密码。
3. 勾选同意隐私政策与使用条款。
4. 点击登录。
5. 进入仪表盘后，先前往“我的课程”同步课程。

## 登录后先做什么

推荐按下面顺序操作：

1. 在“签到规则”里创建至少一条规则。
2. 在“我的课程”里给课程绑定规则。
3. 为需要自动处理的课程开启监控。
4. 需要提醒时，到“通知设置”添加渠道。

::: tip 推荐路径
如果你不确定先点哪个页面，就记住这一条：`登录 -> 同步课程 -> 创建规则 -> 绑定课程 -> 开监控`。
:::

## 仪表盘能看到什么

- 当前已监控课程数量
- 当前待签到任务数量
- 签到成功率
- 最近签到动态

仪表盘适合用来快速确认今天是否有未处理任务。

## 适合新用户的最简流程

如果你只想先跑通一次：

1. 同步课程
2. 新建一条默认规则
3. 给一门课程开启监控
4. 在“我的课程”点击“一键获取任务”
5. 有任务后点击“一键签到”

## 登录失败时先检查什么

- 手机号和密码是否是学习通账号本身的登录信息
- 是否漏勾了隐私政策同意项
- 页面是否要求完成人机验证
- 账号是否被限制使用；如果页面提示冻结或封禁，可按页面入口申诉
