# 通知设置

通知页用于让你更快知道签到是否成功，以及账号是否需要处理。

## 可以添加哪些通知渠道

- 邮件通知
- Webhook
- 微信 Bot

你可以只开一种，也可以按自己的习惯组合使用。

## 邮件通知怎么用

适合想留存结果、方便后续翻看的用户。

常规操作：

1. 在“通知设置”点击“添加渠道”
2. 选择邮件通知
3. 填写邮箱信息
4. 保存后打开启用开关

## Webhook 怎么用

适合需要把签到结果转发到自己的服务、机器人或自动化平台。

操作时主要确认：

- 接收地址是否可用
- 是否需要你自己的下游服务解析消息
- 测试环境和正式环境是否分开

## 微信 Bot 怎么用

适合希望直接在微信里收通知和发简单指令的人。

绑定步骤：

1. 在“通知设置”找到微信 Bot 入口
2. 点击“扫码绑定”
3. 用微信扫描二维码
4. 在手机上确认
5. 绑定成功后，按页面提示发送“帮助”查看可用指令

绑定成功后，常见用途包括：

- 接收签到成功或失败提醒
- 查看个人信息
- 查看我的规则
- 手动触发一些常用操作

## 每个渠道可以控制哪些提醒

你可以分别打开或关闭这些通知项：

- 签到成功
- 签到失败
- 签到列表
- 账号信息

建议至少保留“签到失败”，这样异常时更容易第一时间发现。

## 个人信息页能做什么

通知页里还能顺手看到：

- 当前积分
- 邀请链接
- 一些与账号相关的统计信息

如果你需要邀请别人注册或使用链接进入，这里会比较常用。
