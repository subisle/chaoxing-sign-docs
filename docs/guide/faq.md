# 常见问题

## 为什么我登录后看不到课程

先去“我的课程”点击同步课程。第一次使用时，课程不会自动出现在列表里。

## 为什么课程开启不了监控

通常是因为这门课还没有绑定规则。先创建规则，再回来开启监控。

## 为什么点击一键签到后提示失败

优先检查这几项：

- 有没有给任务选择规则
- 规则里的坐标、图片、链接是否完整
- 当前任务是否需要额外填写二维码链接、签到码或验证码

## 默认规则有什么用

没有单独绑定规则的课程，会优先走默认规则。适合新用户快速起步，也适合统一处理大部分课程。

## 公开分享后，别人会直接拿到我的图片吗

文档使用角度上，你可以把公开分享理解为“分享规则模板”。如果规则里涉及你个人专用内容，分享前建议自行再次确认。

## 微信 Bot 绑定后没反应怎么办

按顺序检查：

1. 二维码是否已经过期
2. 是否已经在手机上确认
3. 渠道开关是否启用
4. 是否关闭了全部通知项

## 学习记录里显示未签到或早退怎么办

先回看对应课程的规则：

- 是否关闭了自动签退
- 是否缺少必要资料
- 是否当时没有开启课程监控

## 后续怎么部署这份文档

1. 推送到 GitHub
2. 连接 StackBlitz 或导入仓库
3. 在文档目录执行安装和启动命令
