# 学习记录

![学习记录界面示意图](/illustrations/notify-history.svg)

学习记录页用于回看签到结果和活动处理情况。

## 签到记录里能看什么

- 哪门课发生了签到
- 成功、失败还是未签到
- 是否涉及签退
- 具体处理时间
- 页面返回的结果说明

如果你想确认“今天到底签没签到成功”，这一页最直接。

## 如何快速筛选

你可以按状态查看：

- 全部
- 成功
- 未签到
- 失败

也可以用搜索框按课程名、签到类型或结果关键字查找。

## 什么时候该重点看学习记录

- 你没收到通知，想补查结果
- 某门课反复失败，想看失败提示
- 想确认签退是否完成
- 想知道某天的签到情况

## 学习活动页

如果当前站点已经开放学习活动功能，记录页里还会看到“学习活动”标签。

你可以在这里查看和处理一些非签到类任务，例如：

- 主题讨论
- 投票
- 问卷
- 评分
- 其他页面已开放的学习活动

是否显示这部分，取决于当前站点是否启用了相关功能。
