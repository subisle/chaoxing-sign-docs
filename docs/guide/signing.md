# 签到与课程

这部分解决两个问题：课程怎么管，签到怎么做。

## 同步课程

第一次进入“我的课程”时，如果列表为空：

1. 点击“立即同步”或“同步课程”
2. 等待课程列表加载完成
3. 检查课程名称、班级和老师信息是否正常

后续如果学期变化或课程有调整，可以再次同步。

## 给课程绑定规则

每门需要自动处理的课程，最好都绑定一条规则。

操作方式：

1. 打开“我的课程”
2. 找到目标课程
3. 选择一条规则
4. 保存绑定结果

如果你已经有默认规则，也建议对重点课程单独指定规则，方便更细地控制。

## 开启课程监控

绑定规则后，打开课程的监控开关，系统才会按你的设置去处理这门课。

你可以：

- 逐门开启
- 使用“全选监控”批量开启
- 不需要时随时关闭

## 一键获取任务

“一键获取任务”适合你主动检查当前是否有新的签到。

常见使用场景：

- 上课前手动确认一次
- 刚收到老师通知，想立刻刷新任务
- 课程列表刚同步完，想马上检查有没有进行中的签到

## 一键签到

发现当前有签到任务后：

1. 点击“一键签到”
2. 勾选要处理的任务
3. 为每个任务确认要用的规则
4. 某些任务需要补充信息时，按页面提示填写
5. 提交并查看结果

## 几种常见签到场景

### 位置签到

- 规则里提前配好坐标
- 任务出现后直接选规则提交

### 二维码签到

- 如果页面要求填写二维码链接，就把对应内容粘贴进去
- 提交前确认匹配的是当前这门课的任务

### 签到码签到

- 已知签到码时，直接填写
- 不知道时，按页面当前能力选择可用操作

### 拍照签到

- 如果规则里已经保存默认图，通常可以直接提交
- 没配图时，建议先补齐规则再用

### 人脸签到

- 必须先在规则里配好人脸照片链接
- 没配好时，任务会提示你当前规则不可直接使用

## 日常推荐用法

### 保守型

1. 先开监控
2. 上课前点一次“一键获取任务”
3. 有任务再手动确认“一键签到”

### 省心型

1. 给常用课程都绑定规则
2. 全部开启监控
3. 配好通知渠道
4. 主要通过通知和学习记录查看结果

## 签到失败后怎么处理

先看结果提示，再按类型排查：

- 规则未选：回到任务里补选规则
- 位置不对：回到规则里修正坐标
- 缺图片：补拍照图或人脸图链接
- 需要验证码：按弹窗提示补输验证码
- 课程没监控：去课程页开启监控
