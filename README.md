# 学习通签到助手文档

这是一个独立的 VitePress 文档仓库，用于提供“学习通签到助手”的用户使用说明。

文档原则：

- 只写功能与使用方法
- 不写实现原理
- 站在终端用户角度组织内容

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## StackBlitz

导入仓库后，在项目根目录执行：

```bash
npm install
npm run dev
```

默认文档目录已经配置好，直接启动即可预览 VitePress 站点。

## 目录说明

- `docs/`：VitePress 文档内容
- `docs/.vitepress/`：站点配置与主题样式
